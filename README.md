# Dolci Sardi Gioiello - Sito Web

Sito web per la pasticceria artigianale "Dolci Sardi Gioiello", specializzata in dolci tipici sardi.

## Caratteristiche

- Design responsivo per tutti i dispositivi
- Navigazione fluida con smooth scroll
- Galleria prodotti
- Sezione storia aziendale
- Contatti e informazioni
- Menu hamburger per dispositivi mobili

## Tecnologie Utilizzate

- HTML5
- CSS3
- JavaScript (ES6+)
- Google Fonts

## Struttura del Sito

1. **Home** - Hero section con presentazione
2. **I Nostri Dolci** - Storia e introduzione
3. **La Nostra Storia** - Timeline dell'azienda
4. **Galleria** - Tutti i prodotti con descrizioni
5. **Contatti** - Informazioni di contatto

## Installazione

1. Clona il repository
2. Apri `index.html` nel browser
3. Oppure visita: https://[tuo-username].github.io/dolci-sardi-gioiello

## Sviluppo

Per modificare il sito:

1. Modifica `index.html` per il contenuto
2. Modifica `style.css` per lo stile
3. Modifica `script.js` per le funzionalità
